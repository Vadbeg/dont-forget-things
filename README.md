# Leaving Home Checklist

Landing page and legal documents for the Leaving Home Checklist iOS app.

## Pages

- **[Landing Page](https://vadbeg.github.io/dont-forget-things/)** - Marketing page with app features and download link
- **[Privacy Policy](https://vadbeg.github.io/dont-forget-things/privacy-policy/)** - Privacy policy for the app
- **[Terms of Use](https://vadbeg.github.io/dont-forget-things/terms-of-use/)** - Terms of use for the app

## Setup

This site is hosted via GitHub Pages. To enable:

1. Go to repository Settings
2. Navigate to Pages section
3. Set source to "Deploy from a branch"
4. Select "main" branch and "/ (root)" folder
5. Save

## Structure

```
/
├── index.html              # Marketing landing page
├── privacy-policy/
│   └── index.html         # Privacy Policy
├── terms-of-use/
│   └── index.html         # Terms of Use
└── assets/
    └── icon.png           # App icon
```

## Updating

After pushing changes to the main branch, GitHub Pages will automatically rebuild and deploy the site.
